# Tiankii Next.js Base Template

This is a base template repository.

## Features
- Next.js (App Router)
- Shadcn UI + Tailwind CSS
- NextAuth configuration
- Custom `apiFetch` wrapper
- Proxy and Middleware configuration

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and fill values
3. `npm run dev`
